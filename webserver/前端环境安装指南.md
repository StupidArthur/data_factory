# 前端环境安装指南

## 环境检查

✅ Node.js 已安装：v24.11.1
✅ npm 已安装：v11.6.2

## 安装步骤

### 1. 创建前端项目

在 `webserver` 目录下执行：

```bash
cd webserver
npm create vite@latest frontend -- --template react
```

**注意**：当提示 `Ok to proceed? (y)` 时，输入 `y` 并按回车。

### 2. 进入前端项目目录

```bash
cd frontend
```

### 3. 安装项目依赖

```bash
npm install
```

### 4. 安装 Ant Design React

```bash
npm install antd
```

### 5. 安装 React Router（用于路由）

```bash
npm install react-router-dom
```

### 6. 安装 Axios（用于API调用）

```bash
npm install axios
```

### 7. 安装开发依赖（可选，用于代码格式化）

```bash
npm install --save-dev eslint prettier
```

## 验证安装

执行以下命令启动开发服务器：

```bash
npm run dev
```

如果看到类似以下输出，说明安装成功：

```
  VITE v5.x.x  ready in xxx ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: use --host to expose
```

## 常见问题

### Q: npm 命令无法执行？
A: 如果遇到 PowerShell 执行策略问题，可以使用：
```bash
cmd /c npm --version
```

或者以管理员身份运行 PowerShell，执行：
```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

### Q: 安装速度慢？
A: 可以使用国内镜像：
```bash
npm config set registry https://registry.npmmirror.com
```

### Q: 端口冲突？
A: 如果 5173 端口被占用，Vite 会自动使用下一个可用端口。
