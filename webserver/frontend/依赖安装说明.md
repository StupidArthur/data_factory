# 前端依赖安装说明

## 需要安装的依赖

图形化配置编辑器需要以下依赖：

```bash
npm install js-yaml
```

## 说明

- `js-yaml`: 用于解析和生成YAML配置文件
- `dayjs`: Ant Design已经内置，无需单独安装

## 安装方法

在 `webserver/frontend` 目录下运行：

```bash
npm install js-yaml
```

如果遇到PowerShell执行策略问题，可以使用以下方法之一：

1. **使用cmd而不是PowerShell**
2. **临时允许脚本执行**：
   ```powershell
   Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope Process
   ```
3. **使用npx**：
   ```powershell
   npx npm install js-yaml
   ```
